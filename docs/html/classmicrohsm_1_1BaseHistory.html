Detailed Description

History base class.

Base class for the DeepHistory and ShallowHistory classes. A history pseudostate is a state that is used for storing the latest shallow or deep configuration of a composite state. When a substate of a composite state is entered, its configuration will be stored in the history psuedostate. When the history pseudostate is entered, it will transition to the latest stored configuration.

A history pseudostate can have a default history. This is the configuration that the HSM will transition when the history's composite state had not been entered before.
