Declare base state A base state can be used as a base class for other states. Normally used to extend the functionality of normal states by adding hooks such as init_. The difference between HSM_DECLARE_STATE and HSM_DECLARE_BASE_STATE is that HSM_DECLARE_BASE_STATE does not declare a match function.
